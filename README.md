# 1. サービス概要
## サービス名：Join us!!(仮)
ほかの候補
- Mochiben(モチベーション+勉強)
- Studius(Study+us=仲間と勉強を連想)
- Manabo(学ぼう)
- StudyMate(Study+Mate=仲間と勉強を連想)

Join us!! はオンライン上で募集をかけて集まった仲間と一緒に学習ができる、一人では学習継続できない人、一緒に頑張る仲間がほしい人のための学習コミュニケーションサービスです。

# 2. このサービスへの思い・作りたい理由
頑張りたいのにひとりでは怠けてしまう。サボってしまう。そんな自分でも、周りに勉強をしている人がいたら頑張れた。そんな気付きから長期間の学習期間でもモチベーションを保つためのサービスがあればいいとおもった。

# 3. ユーザー層について
## **ユーザー層**
- 目標を持って学習しているが、一人だと継続できない人
- プログラミングを学ぶ初学者
- 学習スケジュールや他者との関わりによる｢習慣化の仕組み｣を求めている人
## **理由**
- プログラミングは学習期間が長期間であるため継続するための仕組みが必要
- 一人では学習が進まない人は、同じようなことをしている仲間がいたり、真剣な雰囲気だったりなど環境を整える必要がある。
- 約束(予約)があることで半強制的に学習できる環境を整えられる。

# 4. サービスの利用イメージ
- 事前に学習スケジュールを立ててルームを作成。時間になったら入室し、集まった仲間と一緒に学習。約束によって強制力が働き、集中力が高まる。

# 5. ユーザーの獲得について
- SNSで発信(｢朝活仲間募集｣｢プログラミング初学者｣などハッシュタグをつかう)
- X、Discord,noteのSNSやオンラインスクールコミュニティなどでテストユーザーを募集

# 6. サービスの差別化ポイント・推しポイント
**似たサービス**
- 自習の森:「自習の森」は、勉強仲間で集まり一緒に自習するコミュニティ・社会人サークルです🐰オンライン自習室やカフェ自習会、朝活Zoom自習会など、基本無料で何度でもご参加いただけます☕ 資格勉強・テレワークなど、どのような勉強・作業していただいても大丈夫です。学生・受験生など、社会人以外の方のご参加も歓迎しております🎓

**差別化ポイント**
- 自習の森は、SlackやDiscordなど外部ツールを用いている。一方でJoin us!!はサービス内にビデオ通話やチャット機能が含まれている。そうすることで、通話時間を学習時間として記録したり、学習ごとの時間などのログをつけたりできる。一つのサービスで完結させることでのちのち機能拡張しやすくなるのではと思う。
  - ビデオ通話･チャット･学習記録などを一体化下UIで学習体験をスムーズに
  - SlackやDiscordに頼らず、アプリ内完結で管理・操作がしやすい

# 8. 機能の実装方針予定
## ◆ルーム一覧
- 会員が作成したすべてのルームが表示される。会員はここから参加したいルームを選ぶ。
- 自分が作ったルーム一覧→自分が作成したルームが表示される。ここから削除や編集ができる
- 自分が参加予約しているルームが表示される。ここからルーム参加やルーム詳細、参加辞退(予約削除)ができる

## ◆ルーム作成
- 会員は以下の項目を入力してルームを作成する：
  - ルーム名（例：「朝活・TOEIC勉強会」）
  - ディスクリプション（自由入力）
  - 予定時間（時間指定　例)10:00~16:00）
  - 募集人数（上限指定　例)参加人数1 / 5人※現時点では1on1固定）
  - 作成後、自動的にルーム詳細ページにリダイレクトされる
  - 予定時間まで予約可能。枠が空いていれば予定時刻開始後に即時参加可能
**※作成者は「編集／削除」が可能**

## ◆ルーム詳細
- ルームの概要が表示される
  - 作成者
  - 内容(ルーム名とディスクリプション)
  - 日程
  - 現在の参加者((人数　例)1 / 5名)
- 「参加する」ボタンを押すとルームに入室（即ビデオ通話 or 予定時間待機）
- 作成者は参加者の一覧と退室操作もできるようにする（多人数対応後）

## ◆ルーム参加
- 「参加する」ボタンで参加可能
  - MVPでは 1on1に限定（2人目が参加したら参加ボタンを無効にする）
- 参加後はビデオ通話ルームに接続＋チャット画面が表示される

## ◆ルーム編集
- 作成者のみが編集可能
- 編集可能な項目：タイトル、内容、時間、予定開始時刻 など
- 既に参加者がいる場合、編集後に通知を表示(本リリース後)

## ◆ルーム削除
- 作成者のみが削除可能
- 削除時に「本当に削除しますか？」と確認ダイアログ表示
- 参加中の場合、参加者には「ルームが閉じられました」と通知

## ◆ビデオ通話機能（1on1）
- 参加ボタンを押した2人が通話できるよう、ルームIDを元にWebRTCやSkyWayで接続
- ブラウザ権限（マイク・カメラ）確認あり
- 退出ボタンでビデオ通話を終了し、ルーム一覧に戻る
- 通話にはSkyWayなどWebRTC技術を採用したい。
  - 技術的、時間的な関係でビデオ通話機能は外部ツールを使用する可能性あり。
  - ビデオ通話までアプリ内完結したいが、最優先はコミュニティのための箱を用意すること。

## ◆チャット機能
- ルーム内でリアルタイムにテキストチャットができる
- その他、「メンション」「スタンプ」｢リプライ｣は本リリース

## 本リリース向け機能の方針
- 学習スケジュール機能
  - ルーム作成時、ルーム参加(予約)時にカレンダーへ予定を転記。
  - 学習計画を立てる要領でルーム作成や参加ができる。
  - Xやフレンドに「○○の学習会を開催します」と共有。
- 学習記録機能
  - ルーム滞在時間＝学習時間として自動記録
  - ユーザープロフィールに累計学習時間を表示
  - 「今月○時間集中できました」などモチベ維持になる表示
